AI Shorts Generator

An AI-powered system to automatically generate viral-style short videos using storytelling, voice synthesis, and gameplay footage.

🚀 Features

- AI-generated storytelling scripts (Gen-Z style)
- Text-to-speech voice generation
- Gameplay background integration (Minecraft, Roblox, Fortnite)
- Caption rendering with emoji injection
- Automated video generation using FFmpeg
- Basic trend detection system

🧠 How It Works

1. Generate idea from trends or topic
2. Create script using AI
3. Convert script to voice
4. Select gameplay clip
5. Render video with captions and effects

🏗️ Tech Stack

- Backend: FastAPI (Python)
- Frontend: Next.js
- Video Processing: FFmpeg
- AI: API-based (Gemini / OpenAI)
